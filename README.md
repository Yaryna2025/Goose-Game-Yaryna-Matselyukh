# Flying Goose Game

A simple arcade-style game made with **Python** and **Pygame**. Control your goose, avoid enemies (missiles), and collect bonuses to increase your score.

## Gameplay

- You control the player (a goose) using the arrow keys.
- **Controls:**
  **Arrow Up:**    Move up;
  **Arrow Down:**  Move down;
  **Arrow Right:** Move forward faster;
  **Arrow Left:**  Move backward
- Avoid enemies and collect bonuses to earn points.
- The game ends if you collide with an enemy.
- When the game ends, a **Game Over** screen is displayed.
- Press **any key** on your keyboard to restart the game after a game over.
